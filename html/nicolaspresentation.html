Possibilités :

  • Fermer le jeu avec la X.
  • Quitter le jeu avec 'echap' ou Q
  • Mettre le jeu en pause avec 'espace' ou P
  • Redémarer le jeu avec 'entrée'
  • Cliquer pour mettre un pixel en plus (ou en moins)
